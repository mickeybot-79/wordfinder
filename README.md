# Encuentra Palabras

https://wordfinder-wyhs.onrender.com

Cómo utilizar:
Introducir una serie de letras en el campo "Letras", y un número en el campo "Cantidad".

La idea es encontrar palabras en idioma español que contengan la cantidad total de letras especificada en el campo "Cantidad" y que solamente posean las letras proporcionadas en el campo "Letras", respetando las repeticiones de letras proporcionadas (por ej., si solamente se incluye una "r", ninguna de las palabras resultantes poseerá "rr").

La página consultará una lista de más de 636000 palabras en idioma español, y devolverá solamente aquellas que coincidan con este criterio.

El propósito de esto es hacer trampa en el juego "Words of Wonders".
